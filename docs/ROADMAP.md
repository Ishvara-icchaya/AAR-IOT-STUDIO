# Product roadmap (major v8)

Release **8.0.0** marks the start of a coordinated versioning, lineage, and OTA initiative.

## Phases

| Phase | Objective |
|-------|-----------|
| **0** | Re-enable **Operational lineage** on a **separate screen**, with versioning support aligned to the product model below. |
| **1** | **Role-based views** and **device version metadata**. |
| **2** | **Device version history** and **comparison dashboard**. |
| **3** | **OTA orchestration** framework. |
| **4** | **OTA campaign tracking**, **rollback visibility**, and **audit hardening**. |

Implementation order and scope may be adjusted per delivery; this document tracks the agreed phase intent.

**Normative contract (versioning, schema, pipelines, dashboards):** [DEVICE_VERSIONING_SPEC.md](./DEVICE_VERSIONING_SPEC.md).
